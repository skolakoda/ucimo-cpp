## Kompajliranje

Kompajliranje C++ fajla na Linuxu:

```
g++ zdravo.cpp
```

To će napraviti izlazni fajl `a.out` koji pokrećemo na sledeći način:

```
./a.out
```

Komanda za imenovani izlazni fajl je:

```
g++ zdravo.cpp -o zdravo
```
